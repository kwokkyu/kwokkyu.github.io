PieChart

Overview

  • Implemented a PieChart class which can be instantiated to draw pie charts on the HTML <canvas> elements of web pages

Constructor

PieChart(dataset, color)
  • dataset: an array of objects having name and count properties
  • color: an array of strings representing colors (e.g. color names, hexadecimal codes, RGB functions and values)

The draw() Method

  • Draws slices of the pie chart according to the data in the dataset and colors
  • Utilizes the mathematical sin and cos functions to calculate the positions of the percentage texts and draws them on the pie chart

Responsiveness

  • Made the legend responsive by listening to the resize event of the browser in JavaScript

Usage Example

"use strict";

import { PieChart } from "./piechart.js";

const dataset = [ 
   {name: "Satisfied", count: 1043},
 
   {name: "Neutral", count: 563},
 
   {name: "Unsatisfied", count: 510},
 
   {name: "No comment", count: 175}
 ];

const colors = [ 
   "dodgerblue",
 
   "#FFD700",          // gold
 
   "rgb(255,69,0)",    // orangered
 
   "limegreen"
 ];

let pieChart = new PieChart(dataset, colors);
pieChart.draw();
